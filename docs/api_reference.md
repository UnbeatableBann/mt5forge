# API Reference

Core entry points:

- `TradingEngine`
- `ConfigLoader`
- `MT5Connector`
- `RiskManager`
- `OrderManager`
- `MarketDataFeed`
- `StrategyBase`
- `BacktestRunner`
- `NotificationBus`

All public models are typed dataclasses or Pydantic v2 models and are exported
from their package-level `__init__.py` files.
